安装 openSUSE Tumbleweed

下载工具 和 系统

安装 openSUSE Tumbleweed 可以通过下载并执行安装媒体介质以从头开始安装或者从稳定发行版本升级安装。

从下面下载最新的 Tumbleweed ISO 镜像，使用 ImageWriter 工具将它写入 DVD 或者 U 盘后，在重启要安装的设备开始安装从U盘或DVD开始安装。

ISO 镜像——推荐使用它来进行安装或者升级

DVD 安装镜像 - x86_64

DVD 安装镜像 - i586

准备环境 和 制作 U盘

ImageWriter 工具下载和安装，
    Install ImageWriter for openSUSE （win 版本链接在下边）
    # sudo zypper install imagewriter
Tumbleweed 最新镜像下载备用
用ImageWriter 工具将 Tumbleweed 最新镜像  写入到U盘中。

U盘烧制工具：
ImageWriter for windows 下载
Rufus for windows 下载

安装

F2 选择语言 ， 在点击“安装” 

左侧在列表 一个一个都对号了就可以了。

如果联网了可以点击是，这样可以自动安装为最新的版本。

选择更新用的软件源，点击下一步。

喜欢那个桌面可以选择， 我是KDE，灵活方便自定义。

要注意的就是这个Disk 分区部分， 个人件就用LVM 后期使用方便。  点击 ”引导性配置“  选择 ”LVM卷组模式“

LVM 上层的文件系统都选择 EXT4. 这个后期分区扩缩容方便。  装docker 不要选 btrfs,要不快照会有问题。

时区选择 ”北京时间 或 上海时间“， 图上选择的是上海。

安装过程中 创建一个 用户方便后期使用。
安装配置阅览，这时可以看到安装配置列表中的全部配置，检查好没有问题就可以安装了。


开始安装中。。。。。。。

Language 语言建议装2种。 一个中文，一个英文。 后期可以随时切换。

老的本子装的就是这个系统+KDE5， 用了好久没有出过问题， 是老本子跑不动win 的那种。
