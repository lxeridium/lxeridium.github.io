openSUSE-Tumbleweed (风滚草)

Tumbleweed 项目是一个‘滚动升级’的 openSUSE 版本，俗称‘风滚草’。Tumbleweed 不会像 openSUSE Leap 那样受限于版本开发周期，而是会始终提供最新稳定版本的软件。该项目适合想使用最新的、稳定的软件的 openSUSE 用户。

Tumbleweed 版本基于 openSUSE 的主要开发代码库 Factory。Factory 中的软件被整合、测试后就会进入 Tumbleweed 软件源。Tumbleweed 所含的软件包都是稳定的，可以用于日常使用。

滚动升级这个想法已在邮件列表里讨论了很长一段时间，现在 Greg Kroah-Hartman 将其付诸了行动，详情可以阅读当时的 Tumbleweed 项目公告。然而，当时 Tumbleweed 不是完整的一个发行版而只是一个常规发行版的‘附加产品’。2014 年，Tumbleweed 滚动版和 Factory 滚动版合并，造就了现在的 Tumbleweed。

主要更新：

Linux Kernel 5.8.15、
Mesa 20.2.1、
NodeJS 14.14.0、
Ruby 2.7.2、
NetworkManager 1.26.4、
openSSL 1.1.1h、
KDE Plasma 5.20.1 
cURL 7.73.0 等。。。。

安装 openSUSE Tumbleweed

下载工具 和 系统

安装 openSUSE Tumbleweed 可以通过下载并执行安装媒体介质以从头开始安装或者从稳定发行版本升级安装。

从下面下载最新的 Tumbleweed ISO 镜像，使用 ImageWriter 工具将它写入 DVD 或者 U 盘后，在重启要安装的设备开始安装从U盘或DVD开始安装。

ISO 镜像——推荐使用它来进行安装或者升级

DVD 安装镜像 - x86_64

DVD 安装镜像 - i586

准备环境 和 制作 U盘

ImageWriter 工具下载和安装，
    Install ImageWriter for openSUSE （win 版本链接在下边）
    # sudo zypper install imagewriter
Tumbleweed 最新镜像下载备用
用ImageWriter 工具将 Tumbleweed 最新镜像  写入到U盘中。

U盘烧制工具：
ImageWriter for windows 下载
Rufus for windows 下载

安装

左侧上半部分是 安装配置列表，一项一项完成就可以。 

要注意的就是Disk  分区部分， 个人件就用LVM 后期使用方便。

安装配置阅览，这时可以看到安装配置列表中的全部配置，检查好没有问题就可以安装了。

安装软件，装图形界面，建议KDE5， 后期自定义桌面灵活。其他不懂就默认，后期还是可以安装的。

Language 语言建议装2个。 一个中文，一个英文。 可以随时切换。

我的本子装的就是这个系统+KDE5， 用了好久没有出过问题， 是老本子跑不动win 的那种。
